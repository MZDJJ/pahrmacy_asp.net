# Online Pharmacy Store (pahrmacy)

An integrated dynamic web application for managing an online pharmacy store, built using modern web technologies and secure architecture practices.

## 🚀 Features
* **Product & Inventory Management:** View, add, edit, and delete pharmaceutical products with images, categories, and stock tracking.
* **Order Tracking:** Automated system for logging customer orders, computing totals, and tracking `orderline` details.
* **User & Admin Management:** Secure user accounts system with role-based access control (RBAC) to differentiate between Admins and regular users.
* **Reporting Module:** Generates order summaries and sales reports for pharmacy administrators.

## 🛠️ Tech Stack
* **Backend:** ASP.NET Core / C#
* **Architecture:** MVC (Model-View-Controller) pattern
* **Database:** SQL Server / Entity Framework Core
* **Frontend:** HTML5, Custom CSS3 (Responsive Design), JavaScript
* **Version Control:** Git & GitHub

## 🔒 Security Practices Built-in
* Implemented Role-Based Access Control (RBAC) for sensitive administrative paths (User Management & Reports).
* Configured proper environment management using `.gitignore` to keep configurations and connection strings secure.
